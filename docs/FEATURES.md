# EDU_GUIDE_ENGINE --- FEATURES

## Structured Lesson Builder

Command: /build-lesson

Creates a complete lesson structure from a topic.

Value: reduces teacher planning time ensures clear learning objectives
creates consistent teaching structure

------------------------------------------------------------------------

## Topic Explanation Engine

Command: /explain-topic

Generates layered explanations.

Structure: concept explanation example summary

Value: supports deeper understanding of complex ideas.

------------------------------------------------------------------------

## Practice Generation

Command: /generate-practice

Creates exercises and quizzes.

Types: multiple choice short answer discussion prompts

Value: reinforces knowledge retention.

------------------------------------------------------------------------

## Misconception Detection

Command: /check-misconceptions

Identifies common misunderstandings.

Value: prevents incorrect mental models.

------------------------------------------------------------------------

## Adaptive Explanation System

Command: /adapt-for-level

Rewrites explanations for learner level.

Value: usable across beginner to professional audiences.

------------------------------------------------------------------------
